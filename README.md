# Fraud Academy

Fraud Academy is a React + Vite PWA for practicing realistic fraud and credit risk investigations.

## What is included

- Neon purple/pink investigator dashboard
- Active case queue with generated cases
- Fraud claims separated from credit risk reviews
- Claim types: ATO, email fraud, first party fraud, chargeback claim, and credit risk
- Claim-specific Investigation Toolkit with detailed fictional records
- Claim-specific Document Viewer with fictional evidence documents
- Customer interview transcripts
- Case notes and debrief based on the selected case evidence
- Completed cases remove themselves from the active queue
- Saved progress in the browser with localStorage
- Installable PWA files: manifest, icon, and service worker

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in your terminal.

## Build

```bash
npm run build
npm run preview
```

## Phone install

After deploying the app, open it in Chrome on Android and choose **Add to Home screen** from the browser menu.
